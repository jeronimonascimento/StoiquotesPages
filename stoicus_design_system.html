01

Princípios de Marca

Pilar I
Clareza
O estoicismo elimina o supérfluo. A interface deve respirar. Nada sem propósito permanece na tela.
Pilar II
Profundidade
Cada escolha visual remete à Antiguidade: pergaminho, tinta, ouro. O design carrega peso histórico sem ser pesado.
Pilar III
Serenidade
Nada pisca, nada grita. Animações são suaves e intencionais. O app convida à pausa, não à urgência.
Wordmark
Stoicus
Versão principal — fundo claro
Stoicus
Versão invertida — fundo escuro
Stoicus
Versão dourada — uso especial
Regra: O "us" final é sempre destacado em dourado — representa a conexão entre o legado estoico e o usuário contemporâneo.

Curvas & Durações

500ms · ease · stagger 0.1s

150ms · ease

250ms · ease · auto-dismiss 2.2s
Shimmer
1.4s · linear · loop (skeleton loading)
Princípio: Animações devem sentir-se como folhas caindo — naturais, suaves, sem mecânica artificial. Nunca bounce, nunca elástico. O app convida à pausa.
08

Tom de Voz

Diretrizes de Escrita
✦ Faça
Use presente do indicativo com força: "Entender a reflexão"
Frases curtas, sem verbosidade: "Ver explicação e contexto"
Tom contemplativo, nunca urgente
Labels em maiúsculas esparsas para estrutura: "HOJE · EPICTETO"
Convide à reflexão: "Que escolhas você faria?"
✕ Evite
Exclamações: "Nova reflexão incrível!"
Linguagem de app gamificado: "Ganhe pontos!"
Tecnicidades desnecessárias no UI
Pressão ou urgência: "Compartilhe agora!"
Emojis (exceto em onboarding descontraído)

Tokens CSS

Variáveis — cole no :root do seu projeto
:root {
  /* ── Cores ── */
  --stoic-parchment:       #f5f2eb;  /* bg principal */
  --stoic-parchment-dark:  #ede9de;  /* hover, dividers */
  --stoic-parchment-mid:   #e4dfd3;  /* scrollbar, badges */
  --stoic-ink:             #1a1814;  /* texto principal, bg dark */
  --stoic-ink-mid:         #3d3a33;  /* texto secundário */
  --stoic-ink-light:       #7a7567;  /* labels, meta */
  --stoic-ink-faint:       #b8b2a5;  /* placeholder, hint */
  --stoic-gold:            #c9a84c;  /* accent primário */
  --stoic-gold-light:      #e8d49a;  /* quote marks, ornamentos */
  --stoic-gold-pale:       #f7f0dc;  /* bg tags douradas */
  --stoic-white-card:      #fdfcf9;  /* superfície de cards */

  /* ── Bordas ── */
  --stoic-border:          rgba(26,24,20,0.10);
  --stoic-border-strong:   rgba(26,24,20,0.20);

  /* ── Tipografia ── */
  --stoic-font-display:    'Cormorant Garamond', serif;
  --stoic-font-ui:         'Jost', sans-serif;

  /* ── Radius ── */
  --stoic-radius-tag:      4px;
  --stoic-radius-btn:      10px;
  --stoic-radius-hint:     14px;
  --stoic-radius-card:     20px;

  /* ── Animações ── */
  --stoic-ease:            cubic-bezier(0.25, 0.46, 0.45, 0.94);
  --stoic-duration-fast:   150ms;
  --stoic-duration-base:   300ms;
  --stoic-duration-nav:    450ms;
}
Prefixo --stoic- para evitar conflito em projetos que usem outros design systems (ex: shadcn, Tailwind). Importe Cormorant Garamond e Jost via Google Fonts antes de usar.
